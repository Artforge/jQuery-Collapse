jQuery Collapse

A lightweight (~1kb) jQuery plugin that enables expanding and collapsing content

Collapsible Content with JavaScript

This Plugin delivers an accessible and lightweight solution to a widely adopted interface pattern known as progressive disclosure.

Read the full documentation for more info.

Example

<div id="demo" data-collapse="accordion persist">
  <h2>Fruits</h2>
  <ul>
    <li>Apple</li>
    <li>Pear</li>
    <li>Orange</li>
  </ul>
  <h2>Hint</h2>
  <div>
    <p>One fruit a day keeps the doctor away</p>
  </div>
  <h2>Third</h2>
  <p>Just a paragraph here</p>
</div>

Features

Persistence

Remembers open sections on page reload!

Accordion

The classic 'accordion' behavior of only allowing one open section at a time.

WAI-ARIA Compliance

The plugin has been designed with WAI-ARIA in mind which defines a way to make Web content and Web applications more accessible to people with disabilities.

Lightweight

jQuery-Collapse is designed to be a lightweight plugin that performs well both on small and big scales. It's only ~1kb when compiled and gzipped!

Cross Browser

Fully tested in IE6+, Firefox3+, Chrome5+, Safari4+, Opera 10+. Degrades gracefully in unsupported browsers
